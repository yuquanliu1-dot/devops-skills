Notable shape properties:

- Each entity has 2-7 inbound links from concepts and source summaries
- Each concept has 3-10 inbound links from sources, entities, and other concepts
- The "Disputes" sections on saturated-fat-controversy.md and ultra-processed-foods.md flag real contradictions between source-summary pages
- 4-5 schema updates over 8 months — each one captured a convention that emerged organically
- Lint passes ran ~once per month, each catching 1-3 broken links and 0-2 orphan candidates
- About 10% of query answers got filed back as notes/ pages

What's NOT in this tree:

- A "tag taxonomy" — tags are used loosely in frontmatter, never centralized
- An "outline" or "TOC" page — the index does that work
- Per-source inline images — source images stay in raw/assets/ unless they're worth surfacing
- Generated TOCs or chrome — Markdown editors handle that
