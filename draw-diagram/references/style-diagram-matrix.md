# Style-to-Diagram-Type Adaptation Guide

Not all styles work equally well for every diagram type. Use this guide to pick the best style.

## Architecture Diagram

| Style | Suitability | Notes |
|-------|----------|------|
| 1 Flat Icon | Excellent | Colorful node fills, clear layering |
| 2 Notion Clean | Good | Minimal, works for inline docs |
| 3 Claude Official | Good | Warm aesthetic, Anthropic-style presentations |
| 4 OpenAI Official | Good | Clean, precise; minimal borders, brand green accents |
| 5 Enterprise | Excellent | Best for formal enterprise architecture docs, print-ready |
