# Layout Rules

本文件定义 AI 在生成页面和组件时应遵守的版式规则。

## 1. 统一画布

- 固定使用 `16:9`
- PowerPoint 宽屏坐标按英寸处理
- 建议总画布理解为 `13.333 x 7.5`

## 2. 边距

- 左右安全边距建议不小于 `0.75`
- 顶部安全边距建议不小于 `0.6`
- 底部安全边距建议不小于 `0.5`

## 3. 标题层级

- 一级标题：26 到 32
- 二级标题：18 到 22
- 正文：14 到 20
- 注释：11 到 13

## 4. 文本预算

- 封面标题尽量不超过 18 个汉字
- 单页 bullet 数量尽量不超过 5 个
- 单个 bullet 尽量不超过 24 个汉字
- 普通正文尽量不超过 90 个汉字

## 5. 布局规则

- 封面页优先大标题 + 副标题 + 轻装饰
- 信息页优先标题 + 主体内容 + 辅助视觉
- 图文混排优先两栏，不要随机堆叠
- 表格和图表页必须给足留白

## 6. 一致性规则

- 不要依赖浏览器自动流式排版
- 所有组件必须显式给出 `x/y/w/h`
- 字体、字号、颜色尽量复用主题 token
- 避免使用浏览器特有的复杂 CSS 效果

## 7. 编辑规则

当用户提出自然语言修改请求时：

- 优先改已有组件，不要随意重建整页
- 如果用户要求改布局，先保留内容，再调整组件坐标与尺寸
- 如果用户要求压缩内容，优先删冗余句，不要盲目缩小字号
