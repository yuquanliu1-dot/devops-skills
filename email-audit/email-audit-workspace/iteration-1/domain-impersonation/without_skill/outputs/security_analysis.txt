1. DOMAIN IMPERSONATION (CRITICAL)
   ----------------------------------------------------------------
   From Address: hr@company-intemal.com
                                ^^^^^
   Suspicious Character: The domain uses "intemal" instead of "internal"

   ANALYSIS: The domain "company-intemal.com" contains a typosquatting attack.
   The character sequence "em" in "intemal" visually resembles "rn" or could
   simply be a deliberate misspelling designed to:
   - Pass casual visual inspection
   - Bypass simple email filters that check for "internal" keywords
   - Appear legitimate at first glance

   The legitimate domain would be: company-internal.com
   The malicious domain is: company-intemal.com

   This is a CLASSIC typosquatting/lookalike domain technique.

2. INCONSISTENT CONTACT INFORMATION (HIGH)
   ----------------------------------------------------------------
   From Address: hr@company-intemal.com  (typosquatted domain)
   Body Reference: hr@company-internal.com (legitimate domain mentioned)

   ANALYSIS: The email body references a different domain than the sender's
   address. This inconsistency is a strong indicator of phishing:
   - Attacker made an error by including the legitimate domain in the body
   - This discrepancy should trigger immediate suspicion

3. URGENCY/SOCIAL ENGINEERING (MEDIUM-HIGH)
   ----------------------------------------------------------------
   Language Used: "Action Required" in subject line
                  "must review... by Friday" in body

   ANALYSIS: Classic urgency tactics designed to:
   - Pressure the recipient into acting without careful verification
   - Bypass critical thinking by creating time pressure
   - Exploit employee compliance with HR directives

4. SUSPICIOUS ATTACHMENT (HIGH)
   ----------------------------------------------------------------
   Filename: Employee_Handbook.zip
   Content-Type: application/zip
   Content-Transfer-Encoding: base64

   ANALYSIS: The email contains a ZIP file attachment which is a common
   vector for malware distribution:
   - ZIP files can contain executable payloads
   - Can bypass some email security filters
   - The base64 encoded content suggests actual file data is present
   - The filename "Employee_Handbook.zip" is a common social engineering lure

5. SENDER SPOOFING INDICATORS (MEDIUM)
   ----------------------------------------------------------------
   Display Name: "HR Department"

   ANALYSIS: Using a generic, trusted department name ("HR Department") while
   sending from an external/typosquatted domain is a common impersonation tactic.
